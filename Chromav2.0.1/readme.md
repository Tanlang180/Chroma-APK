Chroma APP 安卓应用软件

- 版本： v2.0.1

- 功能说明
  - Chroma APP 是基于Android平台开发，面向广大自媒体普通用户，为之提供高质量且实时的绿幕视频抠图功能的软件。该软件围绕绿幕视频抠图功能还扩展了背景选择，图像拍照，视频录制，相机翻转等功能，满足了广大自媒体用户日常生活和娱乐的基本需求。
  - 待开发功能
